INSTALO con un PowerShell ```wsl --install``` y para instalar la distribución ubuntu ```wsl --install -d ubuntu```o debian
Cuando lo instalo aparece un dir nuevo en /home/Linux/ y dentro cada uno de los Linux que haya instalado.
Para acceder a la terminal de ubuntu abro un PowerShell y en la pestaña + elijo Ubuntu y ya estoy dentro
Lo compruebo con los comandos:
```
$ pwd
$ lsb_release -a
```

o sobre una PowerShell puedo ver el gestor de máquinas virutales
```
> wsl --list -v
  NAME      STATE           VERSION
* Ubuntu    Running         2
```

Si tengo una versión 2 puedo usar el entorno gráfico 
instalo unas cuantas cosillas:
```
$ sudo apt-get update
$ sudo apt-get install nautilus -y
$ sudo apt-get install octave
```
